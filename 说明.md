# 懂车智析 — 汽车之家口碑采集与智能分析系统 (V2)

## 项目概述

本系统是一个完整的汽车口碑数据分析工具，面向汽车之家网站（k.autohome.com.cn），实现从数据采集、AI智能分析到可视化展示的全流程自动化处理。项目名称"懂车智析"，采用 Flask + Selenium + DeepSeek 大模型 + ECharts 技术栈。

**核心流程：** 输入车型ID → 检索口碑概览 → 爬取全部口碑详情 → 大模型智能分析 → 生成Excel报告与可视化图表。


## 各模块说明

### 1. scraper_engine.py — 数据采集引擎

基于 **Selenium + Chrome headless** 模式，自动爬取汽车之家口碑页面。

- **车型预览：** 输入车型ID，获取车系名称、总口碑数量、在售口碑数量
- **口碑列表爬取：** 自动翻页获取所有口碑详情链接及购车目的
- **口碑详情采集：** 逐条抓取每条口碑的完整信息，包括：
  - 基础信息：车型名称/版本、用户昵称、发表时间、口碑标题
  - 车辆参数：行驶里程、电耗（夏/春/冬）、续航（夏/春/冬）、油耗、裸车价、购买时间/地点
  - 评分信息：空间、驾驶感受、续航、外观、内饰、性价比、智能化、油耗、配置（共9个维度的5星评分）
  - 文字评论：最满意、最不满意，以及各维度详细评论文本
  - 互动数据：观看数、点赞数、评论数
  - 购车目的
- **输出格式：** UTF-8 BOM编码的CSV文件，字段共30+列

### 2. analyzer_engine.py — AI分析引擎

基于 **DeepSeek API（deepseek-chat）** 对爬取的评论文本进行深度分析。

- **评论合并：** 将分散在多个字段（满意/不满意/空间/驾驶感受等）的评论文本拼接为完整评论
- **数据去重：** 基于"评论链接"列自动去除重复记录
- **评分计算：** 根据9个维度评分自动计算综合评分
- **AI分析提取（每条评论）：**
  - **使用场景** — 通勤、接送小孩、自驾游、家庭出行等
  - **满意点** — 满意领域 + 具体描述
  - **不满意点** — 不满意领域 + 具体描述
  - **改进建议** — 改进领域 + 建议内容
  - **对比车型** — 对比车型名称 + 对比内容
- **并发处理：** 使用 ThreadPoolExecutor（5线程）并发调用API，提升分析速度
- **输出格式：** 多Sheet的Excel文件（原始数据 + 使用场景 + 满意点 + 不满意点 + 改进建议 + 对比车型）
- **摘要生成：** 自动生成综合分析摘要（综合评分、最佳/最差维度）

### 3. visualizer_engine.py — 可视化引擎

读取分析结果Excel，进行统计聚类，输出适合前端图表渲染的JSON数据。

- **使用场景聚类：** 用 SequenceMatcher 对场景文本做相似度聚类，输出Top 15
- **领域统计：** 满意领域Top 15、不满意领域Top 15、改进建议领域Top 15
- **对比车型统计：** 用户最常对比的车型Top 15
- **输出格式：** JSON，供 ECharts 饼图渲染

### 4. app.py — Web服务主程序

基于 **Flask** 的Web后端，提供完整的REST API和前端页面。

| API端点 | 方法 | 功能说明 |
|---------|------|---------|
| `/` | GET | 前端界面（懂车智析） |
| `/api/search` | POST | 检索车型口碑概览（预览） |
| `/api/scrape` | POST | 启动异步爬取任务 |
| `/api/analyze` | POST | 启动异步AI分析任务 |
| `/api/visualize` | POST | 生成可视化图表数据 |
| `/api/task/<task_id>` | GET | 查询任务进度 |
| `/outputs/<filename>` | GET | 下载输出文件 |

爬取和分析任务均采用**异步线程**执行，通过轮询 `/api/task/<id>` 获取实时进度。

### 5. templates/index.html — 前端界面

单页Web应用，提供完整的用户交互流程：
1. 输入车型ID → 搜索预览（查看口碑数量）
2. 一键爬取 → 实时进度条
3. 智能分析 → 实时进度条
4. 分析报告展示：综合评分、最满意项、最不满意项、AI摘要
5. 可视化报告：ECharts饼图（使用场景、满意/不满意领域、改进建议、对比车型）
6. 支持导出Excel报告和下载原始CSV数据

## 使用方式

### 环境要求
- Python 3.10+
- Chrome 浏览器 + ChromeDriver
- DeepSeek API Key

### 启动步骤

```bash
# 1. 配置API Key（编辑 .env 文件）
DEEPSEEK_API_KEY=你的DeepSeek_API_Key

# 2. 启动Web服务
python app.py

# 3. 打开浏览器访问
http://localhost:5000
```

### 使用流程
1. 在汽车之家口碑页面找到目标车型ID（URL中的数字，如 `https://k.autohome.com.cn/8171`）
2. 输入车型ID点击"检索口碑"
3. 点击"开始爬取全部口碑"（耗时取决于口碑数量）
4. 爬取完成后点击"前往智能分析"（AI分析耗时取决于评论数量）
5. 查看分析摘要，点击"生成可视化报告"查看图表

## 技术要点

- **反爬对抗：** headless模式 + 自定义User-Agent + 隐藏webdriver特征 + 多种策略提取隐藏交互数据
- **JSON修复：** 大模型返回的JSON可能存在截断，内置括号补齐修复逻辑
- **并发分析：** 5线程并发调用DeepSeek API，配合重试机制和请求延迟
- **文本聚类：** 基于 difflib.SequenceMatcher 的简单相似度聚类，阈值0.55~0.6
- **异步任务：** Flask多线程处理长时间任务，前端轮询获取进度
